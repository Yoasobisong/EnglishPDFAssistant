PDF阅读助手介绍

PDF阅读助手是一款专为英文PDF阅读设计的工具，帮助用户更高效地处理、翻译和学习英文文档。

主要功能

  • PDF转图片 - 将PDF文档转换为带笔记空间的图片，便于阅读和标注
  • 多引擎AI翻译 - 支持多种翻译引擎，智能翻译英文文章
  • 自动词汇提取 - 提取文章中的重要词汇并提供中英文解释
  • 处理结果导出 - 可将翻译、词汇和处理后的图片导出为PDF或文本文件

支持的翻译引擎

引擎名称 特点 适用场景
DeepSeek Reasoner 推理能力强，翻译质量高 学术文献、专业文档
DeepSeek Chat 对话风格自然，速度较快 一般文章、日常阅读
OpenRouter DeepSeek API稳定性好，备选方案 当其他API不可用时

文本提取方法

  • 图像识别 (推荐) - 通过图像处理技术提取文本，适合复杂排版的PDF
  • PyPDF2直接提取 - 直接从PDF文件提取文本，速度快但对复杂排版支持有限
  • PDFPlumber提取 - 更精确的PDF文本提取，适合结构化文档
  • PDFMiner提取 - 强大的PDF解析工具，适合处理复杂的PDF文档
  • OCR识别 - 使用OCR技术识别图像中的文本，适合扫描版PDF

使用须知

  1. 为使用翻译功能，需要在项目根目录的.env文件中配置API密钥
  2. 图像识别和OCR方法在处理大型文档时可能需要较长时间
  3. 翻译和词汇提取依赖于AI服务，需要稳定的网络连接
  4. 处理结果会保存在output目录中，可以随时查看或下载
